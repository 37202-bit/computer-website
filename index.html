องค์ประกอบของระบบคอมพิวเตอร์

ระบบคอมพิวเตอร์ประกอบด้วย 2 ส่วนหลัก ได้แก่ ฮาร์ดแวร์ (Hardware) และซอฟต์แวร์ (Software) ซึ่งทำงานร่วมกันอย่างสมดุลเพื่อให้คอมพิวเตอร์ทำงานได้อย่างมีประสิทธิภาพ

💻 ฮาร์ดแวร์ (Hardware) คืออะไร?

ฮาร์ดแวร์ (Hardware) หมายถึง ส่วนประกอบทางกายภาพของคอมพิวเตอร์ที่สามารถมองเห็นและจับต้องได้จริง ประกอบไปด้วยอุปกรณ์อิเล็กทรอนิกส์ วงจร สายเคเบิล และชิ้นส่วนกลไกต่าง ๆ ที่ทำงานร่วมกัน ฮาร์ดแวร์แบ่งออกเป็น 5 กลุ่มหลักตามหน้าที่ ได้แก่ อุปกรณ์รับเข้า (Input) หน่วยประมวลผล (Processing) หน่วยความจำหลัก (Main Memory) อุปกรณ์จัดเก็บข้อมูล (Storage) และอุปกรณ์แสดงผล (Output) การทำงานของฮาร์ดแวร์ทุกชิ้นจะต้องอาศัยซอฟต์แวร์เป็นตัวสั่งการเพื่อให้คอมพิวเตอร์สามารถทำงานได้ตามวัตถุประสงค์ที่ต้องการ

📥 1.1 อุปกรณ์รับเข้า (Input Devices)

อุปกรณ์รับเข้า คืออุปกรณ์ที่ใช้ในการนำข้อมูลหรือคำสั่งจากผู้ใช้งานเข้าสู่ระบบคอมพิวเตอร์ เพื่อให้คอมพิวเตอร์นำไปประมวลผลต่อไป ข้อมูลที่รับเข้าอาจอยู่ในรูปแบบตัวอักษร ตัวเลข ภาพ เสียง หรือการเคลื่อนไหว

⌨️
Keyboard
แป้นพิมพ์
แป้นพิมพ์เป็นอุปกรณ์รับข้อมูลเข้าหลักที่ใช้กันแพร่หลายที่สุด ประกอบด้วยปุ่มอักขระภาษาอังกฤษ ตัวเลข สัญลักษณ์พิเศษ และปุ่มฟังก์ชัน เมื่อกดปุ่ม วงจรอิเล็กทรอนิกส์ภายในจะสร้างสัญญาณดิจิทัลส่งไปยัง CPU
หลักการทำงาน: เมื่อกดปุ่ม สวิตช์ใต้ปุ่มจะปิดวงจรไฟฟ้า ไมโครคอนโทรลเลอร์ในแป้นพิมพ์จะแปลงสัญญาณเป็นรหัส ASCII หรือ Unicode แล้วส่งผ่านสาย USB หรือ Bluetooth ไปยังคอมพิวเตอร์

ตัวอย่างการใช้งาน: พิมพ์รายงาน ส่งข้อความ กรอกข้อมูล ลัดคีย์บอร์ดในโปรแกรม เล่นเกม และกรอกรหัสผ่าน ปัจจุบันมีแป้นพิมพ์แบบ Mechanical ที่ให้ความรู้สึกกดชัดเจน และแบบ Membrane ที่เงียบและราคาถูกกว่า
🖱️
Mouse
เมาส์ / อุปกรณ์ชี้ตำแหน่ง
เมาส์เป็นอุปกรณ์ควบคุมตำแหน่งของเคอร์เซอร์บนหน้าจอ โดยการเลื่อนเมาส์บนพื้นผิว เมาส์แบบ Optical ใช้แสง LED และกล้องจิ๋วในการตรวจจับการเคลื่อนไหวแทนลูกบอลแบบเก่า
หลักการทำงาน: เซ็นเซอร์ Optical ภายในเมาส์ฉายแสงไปที่พื้นผิวและถ่ายภาพอย่างต่อเนื่อง วิเคราะห์รูปแบบแสงที่สะท้อนกลับมาเพื่อคำนวณทิศทางและความเร็วของการเคลื่อนไหว

ตัวอย่างการใช้งาน: คลิกเลือกไฟล์, ลากและวางวัตถุ, วาดภาพในโปรแกรม Photoshop, เลื่อนหน้าเว็บ, คลิกขวาเพื่อเปิดเมนู เมาส์ Gaming มักมีอัตรา DPI สูง ทำให้ควบคุมได้แม่นยำยิ่งขึ้น
🖨️
Scanner
เครื่องสแกนเนอร์
สแกนเนอร์ทำหน้าที่แปลงเอกสาร รูปภาพ หรือวัตถุ 2 มิติให้กลายเป็นข้อมูลดิจิทัลที่คอมพิวเตอร์สามารถจัดเก็บและประมวลผลได้ ใช้หลักการฉายแสงและตรวจจับแสงสะท้อนผ่านอุปกรณ์ CCD
หลักการทำงาน: หัวสแกนเคลื่อนที่ไปตามเอกสาร ฉายแสงสีขาวลงบนพื้นผิว แสงสะท้อนกลับถูกรับโดยเซ็นเซอร์ CCD (Charge-Coupled Device) แปลงเป็นสัญญาณไฟฟ้า แล้วส่งเป็นข้อมูลดิจิทัล

ตัวอย่างการใช้งาน: สแกนเอกสารราชการ แปลงเอกสารกระดาษเป็น PDF ด้วย OCR สแกนรูปถ่ายเก่าเก็บดิจิทัล สแกนสัญญาธุรกิจ ปัจจุบันสมาร์ทโฟนสามารถสแกนเอกสารผ่านแอปแทนสแกนเนอร์ตั้งโต๊ะได้
📷
Webcam
กล้องเว็บแคม
เว็บแคมเป็นกล้องดิจิทัลที่ต่อเชื่อมกับคอมพิวเตอร์เพื่อบันทึกและส่งภาพวิดีโอแบบเรียลไทม์ผ่านอินเทอร์เน็ต ปัจจุบันมักฝังอยู่ในโน้ตบุ๊กและจอมอนิเตอร์ คุณภาพตั้งแต่ HD ไปจนถึง 4K
หลักการทำงาน: เซ็นเซอร์ภาพ CMOS รับแสงและแปลงเป็นข้อมูลดิจิทัล ชิปประมวลผลภายในจัดการบีบอัดวิดีโอและส่งข้อมูลผ่าน USB หรือสาย AV ไปยังคอมพิวเตอร์อย่างต่อเนื่อง

ตัวอย่างการใช้งาน: ประชุมออนไลน์ผ่าน Zoom/Teams, เรียนออนไลน์, สตรีมสด, บันทึกวิดีโอ vlog, ระบบรักษาความปลอดภัย CCTV และระบบจดจำใบหน้า (Face Recognition) เพื่อเข้าสู่ระบบ
🎤
Microphone
ไมโครโฟน
ไมโครโฟนแปลงคลื่นเสียงในอากาศให้กลายเป็นสัญญาณไฟฟ้า จากนั้น ADC (Analog-to-Digital Converter) แปลงเป็นข้อมูลดิจิทัลที่คอมพิวเตอร์ประมวลผลได้ มีทั้งแบบคอนเดนเซอร์และไดนามิก
หลักการทำงาน: เมมเบรนบาง ๆ ในไมโครโฟนสั่นตามคลื่นเสียง การสั่นนี้สร้างความแตกต่างทางไฟฟ้าซึ่งถูกวัดและแปลงเป็นสัญญาณดิจิทัล ไมค์คอนเดนเซอร์ต้องการไฟ Phantom Power ให้เสียงคมชัดกว่า

ตัวอย่างการใช้งาน: บันทึกเสียงบรรยาย, ประชุมออนไลน์, ส่งคำสั่งเสียงให้ AI เช่น Siri หรือ Google Assistant, อัดพอดแคสต์, Streaming เกม, และใช้งานร่วมกับ Speech-to-Text เพื่อพิมพ์งานด้วยเสียง
📱
Touchscreen
จอสัมผัส
จอสัมผัสรวมหน้าที่ของ Input และ Output Device ไว้ด้วยกัน ผู้ใช้สามารถป้อนข้อมูลโดยตรงด้วยการแตะ ปัดนิ้ว หรือกดบนหน้าจอ ใช้เทคโนโลยี Capacitive ซึ่งตรวจจับกระแสไฟฟ้าจากนิ้วมนุษย์
หลักการทำงาน: หน้าจอ Capacitive Touchscreen เคลือบด้วยวัสดุนำไฟฟ้า เมื่อนิ้วสัมผัส จะรบกวนสนามไฟฟ้าที่จอปล่อยออกมา วงจรตรวจจับตำแหน่งและส่งพิกัด X, Y ไปประมวลผล

ตัวอย่างการใช้งาน: สมาร์ทโฟน แท็บเล็ต คีออสก์บริการตัวเอง จอ ATM เครื่องสั่งอาหาร จอรถยนต์อัจฉริยะ และในงาน Art เช่น iPad + Apple Pencil สำหรับวาดภาพดิจิทัล

⚙️ 1.2 หน่วยประมวลผล (Processing Devices)

หน่วยประมวลผลเป็นศูนย์กลางของระบบคอมพิวเตอร์ ทำหน้าที่รับข้อมูลจาก Input เพื่อประมวลผลตามคำสั่งของโปรแกรม แล้วส่งผลลัพธ์ไปยัง Output หรือหน่วยความจำ

🔲
CPU
Central Processing Unit — หน่วยประมวลผลกลาง
CPU คือ "สมอง" ของคอมพิวเตอร์ ทำหน้าที่อ่านคำสั่งจากโปรแกรม แปลความหมาย และดำเนินการประมวลผลทุกอย่าง ไม่ว่าจะเป็นการคำนวณตัวเลข การจัดการข้อมูล หรือการควบคุมอุปกรณ์ต่อพ่วง วัดความเร็วด้วยหน่วย GHz และจำนวน Core
องค์ประกอบหลัก: ประกอบด้วย Control Unit (CU) ควบคุมการทำงาน, Arithmetic Logic Unit (ALU) คำนวณตัวเลขและตรรกศาสตร์, Registers เก็บข้อมูลชั่วคราว และ Cache Memory ที่ความเร็วสูง

ตัวอย่าง: Intel Core i9, AMD Ryzen 9, Apple M3 Pro มี CPU ทั้ง 4, 8, 16 Core ทำงานพร้อมกันได้หลายงาน ยิ่งมี Core มากและ Clock Speed สูง ยิ่งประมวลผลได้เร็ว
🎮
GPU
Graphics Processing Unit — หน่วยประมวลผลกราฟิก
GPU เป็นชิปประมวลผลพิเศษที่ออกแบบมาเพื่อจัดการกราฟิกและภาพโดยเฉพาะ มีหน่วยประมวลผลขนาดเล็กหลายพันตัวทำงานพร้อมกัน (Parallel Processing) ทำให้เหมาะกับการเรนเดอร์ภาพ 3D วิดีโอ และ AI
หลักการทำงาน: GPU มี Shader Core หลายพันตัวทำงานพร้อมกันในแบบ Parallel ทำให้สามารถคำนวณพิกเซลของภาพ 3D ได้เร็วกว่า CPU มาก มีหน่วยความจำ VRAM แยกต่างหาก

ตัวอย่าง: NVIDIA RTX 4090, AMD Radeon RX 7900 ใช้ในเกม, การเรนเดอร์ 3D ใน Blender, เทรน AI Model ในงาน Deep Learning และ Cryptocurrency Mining
🎛️
Control Unit (CU)
หน่วยควบคุม
Control Unit คือส่วนหนึ่งของ CPU ที่ทำหน้าที่ควบคุมและประสานงานการทำงานของส่วนประกอบต่าง ๆ ในคอมพิวเตอร์ อ่านคำสั่งจาก Memory แปลความหมาย และส่งสัญญาณควบคุมไปยัง ALU และหน่วยต่าง ๆ
หน้าที่: CU ดำเนินการตามวงจร Fetch → Decode → Execute กล่าวคือ ดึงคำสั่ง → ถอดรหัสคำสั่ง → สั่งให้ ALU หรือหน่วยอื่น ๆ ดำเนินการ ทำซ้ำหลายพันล้านครั้งต่อวินาที

ตัวอย่าง: เมื่อผู้ใช้คลิก "Save" CU จะรับคำสั่ง แปลความหมาย แล้วสั่งให้ทั้ง ALU, Memory และ Storage ทำงานร่วมกันเพื่อบันทึกไฟล์ลงดิสก์
🧮
ALU
Arithmetic Logic Unit — หน่วยคำนวณและตรรกศาสตร์
ALU เป็นส่วนของ CPU ที่ทำการคำนวณทางคณิตศาสตร์ (บวก ลบ คูณ หาร) และการดำเนินการทางตรรกศาสตร์ (AND, OR, NOT, XOR) รวมถึงการเปรียบเทียบค่า ซึ่งเป็นพื้นฐานของทุกการประมวลผล
หน้าที่: ALU รับตัวเลข 2 ตัวจาก Register และ Opcode ที่บอกว่าจะทำอะไร เช่น Add, Subtract, Compare แล้วส่งผลลัพธ์กลับไป รองรับทั้งเลขจำนวนเต็มและทศนิยม (Floating Point)

ตัวอย่าง: เมื่อโปรแกรมคำนวณ 100 + 200, CU ส่งคำสั่ง ADD พร้อมค่า 100 และ 200 ให้ ALU ทำการบวก แล้วรับผลลัพธ์ 300 กลับมาเก็บใน Register เพื่อใช้ต่อ
⚡
Cache Memory
หน่วยความจำแคช
Cache เป็นหน่วยความจำความเร็วสูงสุดในระบบ อยู่ภายใน CPU เก็บข้อมูลและคำสั่งที่ CPU ใช้บ่อยเอาไว้ชั่วคราว ทำให้ไม่ต้องเสียเวลาเข้าถึง RAM ทุกครั้ง แบ่งเป็น L1, L2, L3 ตามลำดับความเร็ว
ลำดับ Cache: L1 เล็กที่สุด (~32 KB) แต่เร็วที่สุด, L2 (~256 KB–1 MB), L3 ใหญ่ที่สุด (~8–32 MB) แต่ช้ากว่า L1 ยังไงก็เร็วกว่า RAM มาก โดย Cache L1 เข้าถึงข้อมูลได้ใน ~1–4 Clock Cycle

ตัวอย่าง: เมื่อเปิดเกมครั้งแรก CPU โหลดข้อมูลจาก RAM เข้า Cache ครั้งถัดไปที่ต้องการข้อมูลชุดเดิม CPU เข้าถึงจาก Cache ได้ทันทีโดยไม่ต้องรอ RAM ทำให้เกมไหลลื่น
🖥️
Motherboard
เมนบอร์ด — แผงวงจรหลัก
เมนบอร์ดเป็นแผงวงจรอิเล็กทรอนิกส์หลักที่เชื่อมต่อส่วนประกอบทั้งหมดของคอมพิวเตอร์ไว้ด้วยกัน ทั้ง CPU, RAM, GPU, Storage และ I/O Ports สื่อสารกันผ่าน Bus บนเมนบอร์ด
ส่วนประกอบ: Socket CPU, Slot RAM (DIMM), Slot PCIe สำหรับ GPU, Port SATA และ M.2 สำหรับ Storage, Chip Chipset ควบคุม I/O, BIOS/UEFI ROM และพอร์ต USB, HDMI, Audio

ตัวอย่าง: เมนบอร์ด ASUS ROG, MSI MAG เป็นตัวกลางที่ทำให้ CPU Intel i9 สามารถสื่อสารกับ RAM DDR5, SSD NVMe, GPU NVIDIA และการ์ดเสียงได้พร้อมกัน

🧠 1.3 หน่วยความจำหลัก (Main Memory)

หน่วยความจำหลักทำหน้าที่เก็บข้อมูลและโปรแกรมที่กำลังทำงานอยู่ในขณะนั้น ช่วยให้ CPU เข้าถึงข้อมูลได้รวดเร็ว มี 2 ประเภทหลักคือ RAM และ ROM ซึ่งมีคุณสมบัติแตกต่างกันชัดเจน

💡
RAM
Random Access Memory — หน่วยความจำเข้าถึงสุ่ม
RAM คือหน่วยความจำหลักที่ใช้เก็บข้อมูลและโปรแกรมที่ CPU กำลังทำงานอยู่ในขณะนั้น เป็น Volatile Memory หมายความว่าข้อมูลจะหายไปทันทีเมื่อปิดเครื่อง ยิ่งมี RAM มาก ยิ่งเปิดโปรแกรมพร้อมกันได้มาก
ประเภท: DDR4 และ DDR5 เป็น RAM ยุคปัจจุบัน DDR5 เร็วและประหยัดพลังงานกว่า ความจุทั่วไปคือ 8–64 GB ความเร็ว 3200–6400 MHz

ตัวอย่าง: เปิด Chrome 20 แท็บ + VS Code + Zoom พร้อมกัน ต้องใช้ RAM อย่างน้อย 16 GB มิฉะนั้นระบบจะช้าเพราะต้องใช้ Virtual Memory บน Disk แทน
🔒
ROM
Read-Only Memory — หน่วยความจำอ่านอย่างเดียว
ROM เป็นหน่วยความจำถาวร (Non-Volatile) ที่เก็บโปรแกรมพื้นฐานของระบบ เช่น BIOS/UEFI ข้อมูลใน ROM ไม่สูญหายเมื่อปิดเครื่อง และโดยปกติผู้ใช้จะไม่สามารถแก้ไขได้ในระหว่างการใช้งานปกติ
ประเภท: PROM (เขียนได้ครั้งเดียว), EPROM (ลบได้ด้วยแสง UV), EEPROM (ลบได้ด้วยสัญญาณไฟฟ้า), Flash ROM (ปัจจุบันใช้ใน BIOS/UEFI)

ตัวอย่าง: เมื่อเปิดคอมพิวเตอร์ BIOS ใน ROM จะตรวจสอบฮาร์ดแวร์ (POST) และบูตระบบปฏิบัติการขึ้นมาก่อนเสมอ โดยไม่ต้องพึ่ง RAM
🔄
Virtual Memory
หน่วยความจำเสมือน
Virtual Memory คือเทคนิคที่ระบบปฏิบัติการใช้พื้นที่บน Hard Disk หรือ SSD มาทำงานแทน RAM เมื่อ RAM ไม่เพียงพอ ทำให้ระบบยังทำงานต่อได้ แต่จะช้ากว่า RAM จริงมาก
หลักการ: OS สร้างไฟล์ Paging (pagefile.sys บน Windows หรือ Swap บน Linux) บน Disk ข้อมูลที่ไม่ได้ใช้บ่อยจะถูก "Swap Out" ไปเก็บใน Disk และโหลดกลับเข้า RAM เมื่อต้องการ

ตัวอย่าง: คอมพิวเตอร์ RAM 4 GB เปิดโปรแกรมหนักหลายตัว Windows จะใช้ Paging File บน SSD เป็น Virtual Memory เพิ่มเติม แต่จะรู้สึกช้าลงอย่างเห็นได้ชัด
ประเภท ความเร็ว ความจุทั่วไป ถาวร/ชั่วคราว หน้าที่
Cache (L1) เร็วมาก 32–64 KB ชั่วคราว เก็บข้อมูลที่ CPU ใช้บ่อยที่สุด
Cache (L3) เร็วมาก 8–32 MB ชั่วคราว บัฟเฟอร์ระหว่าง CPU กับ RAM
RAM (DDR5) ปานกลาง 8–64 GB ชั่วคราว เก็บโปรแกรมที่กำลังทำงาน
ROM (BIOS) ปานกลาง 16–128 MB ถาวร เก็บ Firmware บูตระบบ
Virtual Memory ช้า แล้วแต่ Disk ชั่วคราว ขยาย RAM เมื่อไม่เพียงพอ

💽 1.4 อุปกรณ์จัดเก็บข้อมูล (Secondary Storage)

อุปกรณ์จัดเก็บข้อมูลสำรองทำหน้าที่เก็บข้อมูลอย่างถาวร ข้อมูลจะยังคงอยู่แม้จะปิดเครื่อง แตกต่างจาก RAM ที่ข้อมูลหายเมื่อหยุดจ่ายไฟ ปัจจุบันมีหลายรูปแบบที่แตกต่างกันในด้านความเร็ว ความจุ และราคา

💿
HDD
Hard Disk Drive — ฮาร์ดดิสก์
HDD ใช้จานแม่เหล็กหมุนเก็บข้อมูล มีหัวอ่าน/เขียนที่เคลื่อนที่ข้ามจาน ความจุสูงสุดถึง 20 TB ราคาถูกต่อ GB แต่ช้ากว่า SSD มากเนื่องจากมีชิ้นส่วนเคลื่อนไหวทางกลไก
ข้อดี: ราคาถูก ความจุสูง เหมาะเก็บไฟล์ขนาดใหญ่ เช่น ภาพยนตร์ เพลง White space ขนาดใหญ่

ข้อเสีย: ช้ากว่า SSD ประมาณ 5–10 เท่า เสียหายได้ง่ายจากการกระแทก เนื่องจากหัวอ่านลอยอยู่เหนือจานแค่นาโนเมตร

ตัวอย่าง: Seagate Barracuda 2TB, WD Blue ราคาประมาณ 1,500 บาท ต่อ 2TB เหมาะสำหรับ NAS เก็บข้อมูล
⚡
SSD
Solid State Drive — เอสเอสดี
SSD ใช้ชิป Flash Memory เก็บข้อมูล ไม่มีชิ้นส่วนเคลื่อนไหว ทำให้เร็วกว่า HDD 5–20 เท่า ทนทานต่อการกระแทก เงียบ และประหยัดพลังงาน ปัจจุบัน SSD แบบ NVMe ผ่าน PCIe เร็วกว่า SATA SSD อีก 5 เท่า
ประเภท: SATA SSD (อ่าน ~550 MB/s), NVMe M.2 (อ่าน ~3,500–7,000 MB/s), PCIe 5.0 NVMe รุ่นใหม่สุดเร็วถึง 12,000 MB/s

ตัวอย่าง: Samsung 990 Pro 1TB NVMe เป็น SSD ยอดนิยม บูต Windows ใน 10–15 วินาที เปิดโปรแกรมได้ทันที เหมาะเป็น System Drive สำหรับนักเรียนและมืออาชีพ
🔌
USB Flash Drive
แฟลชไดรว์ / Thumb Drive
แฟลชไดรว์เป็นอุปกรณ์เก็บข้อมูลแบบพกพาขนาดเล็กที่ใช้ Flash Memory เชื่อมต่อผ่าน USB ใช้งานง่าย ไม่ต้องติดตั้งไดรเวอร์ รองรับการเสียบและถอด (Plug and Play) ความจุ 4 GB ถึง 1 TB
ข้อดี: พกพาสะดวก ราคาถูก ใช้ได้กับอุปกรณ์ทุกชนิดที่มี USB Port ทนทาน ใช้ได้ทั้ง Windows/Mac/Linux

ตัวอย่าง: นักเรียนใช้แฟลชไดรว์ส่งงาน โอนไฟล์นำเสนอ USB 3.2 Gen 2 ความเร็วสูงถึง 10 Gbps เหมาะโอนไฟล์ขนาดใหญ่ SanDisk, Kingston เป็นยี่ห้อที่นิยม
🗂️
Memory Card
เมมโมรี่การ์ด
Memory Card เป็น Flash Memory ขนาดเล็กในรูปแบบการ์ด ใช้ใน กล้องถ่ายภาพ สมาร์ทโฟน โดรน เครื่องเล่นเกม มาตรฐาน SD ที่นิยมได้แก่ SD, microSD, SDXC, CFexpress สำหรับมืออาชีพ
คลาสความเร็ว: Class 10 (10 MB/s ขึ้นไป), UHS-I (104 MB/s), UHS-II (312 MB/s), V90 (90 MB/s เขียน) เหมาะกับวิดีโอ 4K/8K

ตัวอย่าง: Samsung Pro Plus microSD 256GB เหมาะสำหรับ Nintendo Switch เก็บเกมและถ่ายวิดีโอ 4K บนกล้อง Sony A7 ได้ต่อเนื่อง
☁️
Cloud Storage
ที่เก็บข้อมูลบนคลาวด์
Cloud Storage เก็บข้อมูลบน Server ของผู้ให้บริการที่เชื่อมต่อผ่านอินเทอร์เน็ต เข้าถึงได้จากทุกที่ทุกเวลาผ่านทุกอุปกรณ์ ปลอดภัยจากภัยพิบัติ มีระบบสำรองข้อมูลอัตโนมัติ
ตัวอย่างบริการ: Google Drive (15 GB ฟรี), OneDrive (5 GB ฟรี), iCloud, Dropbox, MEGA (20 GB ฟรี) มีทั้งแผนฟรีและแบบชำระเงิน

การใช้งาน: นักเรียนบันทึกรายงานบน Google Drive แชร์ให้เพื่อนร่วมแก้ไขพร้อมกันแบบ Real-time, ธุรกิจสำรองข้อมูลสำคัญ, Photographer อัปโหลดรูปจากกล้องสู่ Adobe Lightroom
🔋
External HDD
ฮาร์ดดิสก์ภายนอก
External HDD คือฮาร์ดดิสก์ที่บรรจุในกล่องพกพาเชื่อมต่อด้วย USB เป็นอุปกรณ์สำรองข้อมูลและพกพาข้อมูลจำนวนมาก ความจุสูงถึง 20 TB ราคาไม่แพง เหมาะสำรองข้อมูลสำคัญ
ตัวอย่าง: WD My Passport 2TB, Seagate Backup Plus เหมาะสำรองระบบด้วย Windows Backup หรือ Time Machine บน Mac ตามกฎ 3-2-1 Backup

ข้อควรระวัง: ต้องป้องกันการกระแทก เพราะมีจานแม่เหล็กหมุนอยู่ภายใน ปัจจุบัน External SSD เช่น Samsung T7 ทนทานกว่าและเร็วกว่า แต่ราคาสูงกว่า

📤 1.5 อุปกรณ์แสดงผล (Output Devices)

อุปกรณ์แสดงผลทำหน้าที่นำข้อมูลที่คอมพิวเตอร์ประมวลผลเสร็จแล้วออกมาให้ผู้ใช้รับรู้ได้ ทั้งในรูปแบบภาพ เสียง ข้อความ หรือสิ่งพิมพ์

🖥️
Monitor
จอภาพ / จอมอนิเตอร์
จอมอนิเตอร์เป็น Output Device หลักที่แสดงผลในรูปภาพและข้อความ ประเภทปัจจุบันได้แก่ IPS, VA, OLED มีความละเอียด FHD, QHD, 4K และอัตราการรีเฟรช 60–360 Hz สำหรับเกมเมอร์
เทคโนโลยีหลัก: IPS (สีสวย มุมมองกว้าง), VA (คอนทราสต์สูง), OLED (ดำจริง สีสด แต่ burn-in), Mini-LED สำหรับ HDR คุณภาพสูง

ตัวอย่าง: นักออกแบบใช้ DELL U2723D 4K IPS เพื่อความแม่นยำสี นักเล่นเกมใช้ ASUS ROG Swift 360 Hz เพื่อ Response Time ต่ำ
🖨️
Printer
เครื่องพิมพ์
Printer แปลงไฟล์ดิจิทัลเป็นสิ่งพิมพ์บนกระดาษหรือวัสดุอื่น ๆ มีทั้งแบบ Inkjet ที่ราคาถูกใช้หมึก, Laser ที่เร็วและคมชัดสำหรับเอกสาร, และ 3D Printer ที่พิมพ์วัตถุสามมิติ
ประเภท: Inkjet (Epson, Canon) เหมาะพิมพ์รูป, Laser (HP, Brother) เหมาะพิมพ์เอกสารปริมาณมาก, Thermal ใช้ใบเสร็จ, 3D Printer (FDM, Resin) สร้างชิ้นงาน 3 มิติ

ตัวอย่าง: โรงเรียนใช้ Laser Printer พิมพ์ข้อสอบ, นักออกแบบใช้ Inkjet พิมพ์รูปคุณภาพสูง, วิศวกรใช้ 3D Printer สร้าง Prototype ชิ้นงาน
🔊
Speaker
ลำโพง
ลำโพงแปลงสัญญาณไฟฟ้าเป็นคลื่นเสียงที่หูมนุษย์ได้ยิน การ์ดเสียงในคอมพิวเตอร์ส่งสัญญาณ Analog ไปขับ Driver ในลำโพงสร้างการสั่นสะเทือนในอากาศ ความถี่ 20 Hz–20 kHz
ตัวอย่าง: Logitech Z623 ลำโพง 2.1 เหมาะเกมและหนัง, Edifier ลำโพง Studio Monitor เหมาะ Home Studio, JBL, Harman สำหรับดนตรีคุณภาพ

ระบบ Surround: 2.0 (สเตอริโอ), 2.1 (ซ้าย-ขวา+ซับ), 5.1 (โฮมเธียเตอร์), 7.1.4 (Dolby Atmos) สำหรับประสบการณ์เสียงรอบทิศทาง
📽️
Projector
โปรเจกเตอร์
โปรเจกเตอร์ฉายภาพจากคอมพิวเตอร์ขึ้นบนผนังหรือจอรับภาพขนาดใหญ่ เหมาะสำหรับห้องเรียน ห้องประชุม และโรงภาพยนตร์ เทคโนโลยี DLP และ LCD ให้ภาพสว่างคมชัด
ตัวอย่าง: Epson EB-X51 ใช้ในห้องเรียน, BenQ FullHD สำหรับโฮมเธียเตอร์, NEC PA804UL เลเซอร์ 8,000 lumens ใช้ในหอประชุมใหญ่

เทคโนโลยี: Laser Projector ไม่ต้องเปลี่ยนหลอด อายุการใช้งาน 20,000+ ชั่วโมง Short Throw Projector ฉายจากระยะใกล้เหมาะห้องแคบ
📺
Smart TV
โทรทัศน์อัจฉริยะ
Smart TV คือโทรทัศน์ที่มีระบบปฏิบัติการในตัว เช่น Android TV, webOS, Tizen สามารถรับสัญญาณจากคอมพิวเตอร์ผ่าน HDMI, Miracast, AirPlay และเล่น Netflix, YouTube โดยตรง
ตัวอย่าง: Samsung Neo QLED, LG OLED evo เชื่อมต่อกับคอมพิวเตอร์เพื่อนำเสนองาน เล่นเกม PC บนจอใหญ่ หรือใช้เป็น Second Monitor ในการทำงาน

ข้อดีในการเรียน: ใช้เป็นจอนำเสนอในห้องเรียน เชื่อมต่อไร้สายจากแท็บเล็ต/โน้ตบุ๊กของครู สร้างประสบการณ์เรียนรู้ที่น่าสนใจ
🎧
Headphone
หูฟัง
หูฟังเป็น Output Device เฉพาะบุคคล แปลงสัญญาณไฟฟ้าเป็นเสียงที่ส่งตรงเข้าหู มีทั้งแบบมีสาย (3.5mm, USB-C) และไร้สาย (Bluetooth) พร้อมฟีเจอร์ ANC (Active Noise Cancellation)
ตัวอย่าง: Sony WH-1000XM5, Apple AirPods Pro มี ANC ตัดเสียงรบกวน เหมาะทำงานในที่พลุกพล่าน, Sennheiser HD 800S เหมาะ Audiophile ฟังเพลงคุณภาพสูง

ใน Gaming: Headset เพื่อเกม เช่น Razer BlackShark มีไมค์ในตัว ระบบ Surround 7.1 ได้ยินทิศทางเสียงในเกม สำคัญมากใน FPS เช่น CS2, Valorant

💾 ซอฟต์แวร์ (Software) คืออะไร?

ซอฟต์แวร์ (Software) หมายถึง ชุดคำสั่งหรือโปรแกรมที่เขียนขึ้นด้วยภาษาคอมพิวเตอร์ เพื่อสั่งให้ฮาร์ดแวร์ทำงานตามวัตถุประสงค์ที่ต้องการ ซอฟต์แวร์ไม่มีตัวตนที่จับต้องได้ แต่เป็นส่วนที่ขาดไม่ได้ในการทำให้คอมพิวเตอร์มีประโยชน์ แบ่งเป็น 2 ประเภทหลัก คือ ซอฟต์แวร์ระบบ (System Software) ที่ทำงานเป็นพื้นฐานของระบบ และซอฟต์แวร์ประยุกต์ (Application Software) ที่ผู้ใช้ใช้โดยตรงเพื่อวัตถุประสงค์เฉพาะ

🔧 2.1 System Software (ซอฟต์แวร์ระบบ)

🪟
Operating System
ระบบปฏิบัติการ (OS)
OS คือซอฟต์แวร์ที่จัดการทรัพยากรฮาร์ดแวร์ทั้งหมด ได้แก่ CPU, RAM, Storage, I/O และควบคุมให้โปรแกรมประยุกต์ทำงานบนระบบได้อย่างราบรื่น OS ทำหน้าที่เป็นตัวกลางระหว่างผู้ใช้กับฮาร์ดแวร์
ตัวอย่าง OS: Windows 11 (PC ทั่วไป), macOS Sonoma (Apple), Ubuntu/Linux (Server/Developer), Android (สมาร์ทโฟน), iOS (iPhone) แต่ละตัวมีจุดเด่นต่างกัน

หน้าที่หลัก: จัดการไฟล์ (File System), จัดสรร RAM ให้โปรแกรม, ควบคุม Driver, UI (GUI/CLI), รักษาความปลอดภัย, เชื่อมต่อเครือข่าย
🔌
Device Driver
ไดรเวอร์อุปกรณ์
Device Driver คือโปรแกรมขนาดเล็กที่ทำหน้าที่แปลคำสั่งจาก OS ให้อุปกรณ์ฮาร์ดแวร์เข้าใจและทำงานได้ถูกต้อง ทุกอุปกรณ์ที่ต่อเข้าคอมพิวเตอร์ต้องมี Driver ที่เหมาะสม
ตัวอย่าง: NVIDIA GeForce Driver สำหรับ GPU, Realtek Audio Driver สำหรับเสียง, Driver เครื่องพิมพ์จาก Canon/Epson Windows Update มักอัปเดต Driver อัตโนมัติ

ทำไมสำคัญ: Driver เก่าหรือผิดรุ่นทำให้อุปกรณ์ทำงานผิดปกติ ช้า หรือ Crash ควรอัปเดต Driver GPU เป็นประจำเพื่อประสิทธิภาพเกมและ AI สูงสุด
🛠️
Utility Program
โปรแกรมอรรถประโยชน์
Utility Program ช่วยดูแลและบำรุงรักษาระบบคอมพิวเตอร์ให้ทำงานได้อย่างมีประสิทธิภาพ เช่น โปรแกรม Defragment ดิสก์ โปรแกรมบีบอัดไฟล์ โปรแกรมสำรองข้อมูล และ Task Manager
ตัวอย่าง: WinRAR/7-Zip บีบอัดไฟล์, CCleaner ลบไฟล์ขยะ, Disk Cleanup, Task Manager ดู CPU/RAM, HWMonitor วัดอุณหภูมิ, Macrium Reflect สำรองระบบ

ความสำคัญ: Utility ช่วยให้ระบบทำงานเร็วขึ้น ป้องกันปัญหา ควรรันโปรแกรมทำความสะอาดและสแกนมัลแวร์สม่ำเสมอ
💽
Firmware
เฟิร์มแวร์
Firmware คือซอฟต์แวร์ที่บันทึกถาวรลงในชิปของฮาร์ดแวร์ ควบคุมการทำงานขั้นพื้นฐานที่สุดของอุปกรณ์นั้น เช่น BIOS/UEFI ในเมนบอร์ด Firmware กล้องถ่ายภาพ หรือ Firmware เราเตอร์
ตัวอย่าง: BIOS/UEFI บูตระบบ, Firmware SSD ปรับปรุงประสิทธิภาพ, Firmware เราเตอร์แก้ช่องโหว่ความปลอดภัย, Firmware กล้อง เพิ่มฟีเจอร์ใหม่

การอัปเดต: ควรอัปเดต Firmware อุปกรณ์สำคัญเป็นระยะ เพราะผู้ผลิตมักออกแพตช์แก้บัค เพิ่มฟีเจอร์ และปิดช่องโหว่ความปลอดภัย
🛡️
Antivirus
โปรแกรมป้องกันไวรัส
Antivirus ทำหน้าที่ตรวจจับ ป้องกัน และลบมัลแวร์ทุกประเภท ได้แก่ ไวรัส Trojan Ransomware Spyware และ Phishing ใช้ฐานข้อมูล Signature และ AI วิเคราะห์พฤติกรรมที่น่าสงสัย
ตัวอย่าง: Windows Defender (ฟรีใน Windows), Kaspersky, Bitdefender, Malwarebytes ควรมี Antivirus ที่ดีและอัปเดต Definition ทุกวัน

ภัยคุกคามสมัยใหม่: Ransomware เข้ารหัสไฟล์เรียกค่าไถ่, Phishing หลอกกรอกรหัสผ่าน, Keylogger บันทึกการพิมพ์ ควรใช้ทั้ง Antivirus และ VPN เพื่อความปลอดภัยสูงสุด

ส่วนที่ 2

หลักการทำงานของคอมพิวเตอร์

คอมพิวเตอร์ทำงานตามวงจร Input → Process → Memory → Storage → Output กดที่แต่ละขั้นตอนเพื่อดูรายละเอียด

📥
① INPUT — รับข้อมูลเข้า
ผู้ใช้ป้อนข้อมูลหรือคำสั่งผ่านอุปกรณ์ Input เช่น แป้นพิมพ์ เมาส์ ไมโครโฟน กล้อง หรือหน้าจอสัมผัส ข้อมูลจะถูกแปลงเป็นสัญญาณดิจิทัล (0 และ 1) เพื่อส่งต่อให้ CPU ประมวลผล

🔍 ตัวอย่างการรับข้อมูล

  • พิมพ์งาน: กดแป้นพิมพ์ → สัญญาณ ASCII → ส่งผ่าน USB → CPU
  • เล่นเกม: กดปุ่ม Controller → สัญญาณ Bluetooth → CPU ประมวลผลตำแหน่งตัวละคร
  • เปิดเว็บ: พิมพ์ URL → ส่งผ่าน Network Card → DNS Lookup → เชื่อมต่อ Server
  • ดูวิดีโอ: คลิก Play → ส่งคำสั่งไป Media Player → อ่านไฟล์จาก Storage
⌨️ แป้นพิมพ์ 🖱️ เมาส์ 🎤 เสียง 📷 กล้อง 📱 สัมผัส
⚙️
② PROCESS — ประมวลผล
CPU รับข้อมูลจาก Input และ Memory มาประมวลผลตามคำสั่งของโปรแกรม โดยผ่านวงจร Fetch → Decode → Execute ซ้ำหลายพันล้านครั้งต่อวินาที CU ควบคุม ALU คำนวณ และ Cache เก็บข้อมูลชั่วคราว

⚙️ วงจรการประมวลผล (Fetch-Decode-Execute)

① FETCH
CU ดึงคำสั่งถัดไปจาก RAM ผ่าน Memory Bus เก็บใน Instruction Register
② DECODE
CU ถอดรหัสคำสั่ง แปลง Opcode ว่าจะให้ทำอะไร กับข้อมูลใด
③ EXECUTE
ALU หรือหน่วยอื่นดำเนินการคำสั่ง เก็บผลลัพธ์ใน Register
🔲 CPU Core 🧮 ALU 🎛️ Control Unit ⚡ Cache L1/L2/L3
💡
③ MAIN MEMORY — หน่วยความจำหลัก
ขณะที่โปรแกรมทำงาน CPU จะโหลดข้อมูลและคำสั่งที่จำเป็นจาก Storage เข้าสู่ RAM เพื่อให้เข้าถึงได้รวดเร็ว RAM ทำหน้าที่เป็น "โต๊ะทำงาน" ชั่วคราวของ CPU ยิ่งมี RAM มาก ยิ่งเปิดโปรแกรมพร้อมกันได้มาก

💡 การทำงานของ RAM

เมื่อดับเบิลคลิกไอคอนโปรแกรม OS จะอ่านไฟล์จาก SSD/HDD แล้วโหลดเข้า RAM จากนั้น CPU จะอ่านคำสั่งจาก RAM ทีละคำสั่งเพื่อประมวลผล ข้อมูลที่ใช้บ่อยจะถูกคัดลอกขึ้นไปเก็บใน Cache ของ CPU เพื่อความเร็วสูงสุด

💾 RAM DDR5 🔒 ROM BIOS ⚡ Cache 🔄 Virtual Memory
💽
④ STORAGE — จัดเก็บข้อมูล
เมื่อผู้ใช้สั่ง "บันทึก" ข้อมูลจาก RAM จะถูกเขียนลง Storage (SSD/HDD) เพื่อเก็บอย่างถาวร ข้อมูลใน Storage จะคงอยู่แม้ปิดเครื่อง ในทางกลับกัน เมื่อเปิดไฟล์ OS จะอ่านข้อมูลจาก Storage โหลดขึ้น RAM

💽 วงจรการจัดเก็บ

บันทึกไฟล์: กด Ctrl+S → OS รับคำสั่ง → เขียนข้อมูลจาก RAM ลง SSD → ยืนยันว่าบันทึกสำเร็จ

เปิดไฟล์: ดับเบิลคลิก → OS อ่านจาก SSD → โหลดเข้า RAM → CPU ประมวลผล → แสดงผลบนหน้าจอ

💿 SSD NVMe 🔋 HDD ☁️ Cloud 🔌 USB Drive
📤
⑤ OUTPUT — แสดงผลลัพธ์
ผลลัพธ์จากการประมวลผลจะถูกส่งออกผ่าน Output Device เพื่อให้ผู้ใช้รับรู้ได้ ไม่ว่าจะเป็นภาพบนจอ เสียงจากลำโพง เอกสารจากเครื่องพิมพ์ หรือสัญญาณส่งผ่านเครือข่ายอินเทอร์เน็ต

📤 ตัวอย่าง Output ในชีวิตจริง

  • พิมพ์งาน: ตัวอักษรปรากฏบนจอ → กด Print → Printer พิมพ์กระดาษออกมา
  • เล่นเกม: GPU เรนเดอร์ภาพ → ส่งสัญญาณ HDMI → Monitor แสดง 144 fps
  • ฟังเพลง: Decode MP3 → Sound Card แปลงเป็น Analog → ลำโพงสร้างเสียง
  • ส่ง Email: ข้อมูลออกจาก CPU → Network Card → Router → Internet → Server ปลายทาง
🖥️ Monitor 🔊 Speaker 🖨️ Printer 📡 Network
